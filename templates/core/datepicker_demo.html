HTML Usage

JavaScript Usage
// Auto-initialization
// All date inputs are automatically converted

// Manual initialization
new MilitaryDatePicker(input, {
    format: 'yyyy-mm-dd',
    showTime: true,
    quickDates: true,
    minDate: new Date(),
    maxDate: new Date(2025, 11, 31)
});
